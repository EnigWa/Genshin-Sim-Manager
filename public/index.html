How to Use

1. Auto-managed gcsim binary
This App Auto Detects your OS and downloads the relevant binary from GCsim's Github page.

2. Create a project
Click Create and give it a name to organise your configs.

3. Add configs
Click + New in the sidebar to create config files, then write or paste your sim config in the editor.

4. Compare Multiple Configs
Hover over a config in the sidebar and click the ◫ button to open it in a secondary pane side-by-side.

5. Run or Optimize
Select a config in the sidebar and click Run or Optimize. Use Stop to cancel.

Results will seamlessly and automatically open a browser window displaying the gcsim viewer UI.
